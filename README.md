# every-embodied-task

## every-embodied项目学习指南

开源内容Github仓库：https://github.com/datawhalechina/every-embodied

学习文档：https://datawhaler.feishu.cn/wiki/NeCKwDHmFiPZHwkhM3VcHrDenzg

收集到学习者的问题反馈，在进行实验的时候windows系统不适配，启用子系统WSL2仍然会出现很多问题，在无显示的服务器上容易出现报错且无法看到仿真模拟等。基于以上问题，安装双系统或使用云服务器是两种可实现的方式。

本项目主要是方便学习者复现，使用双系统和云服务器并行的方式进行，供大家学习参考，具体学习内容可以参考上述仓库和文档。

## 使用方法

```bash
git clone https://github.com/Ethanwhh/every-embodied-task.git
```

每个项目的使用说明见每个项目的说明文档，如  Task01/task01.md
